遗留问题

  1. Markdown PDF没有办法解决$\LaTeX$公式的渲染问题，暂时只能使用Markdown Preview Enhance来导出到html再另存为pdf格式。
  2. Markdown Preview Enhance也是可以配置css文件的。
  3. Markdown Preview Enhance可以配置多种流程图，但是渲染等需要再研究下。
  4. 关于$\LaTeX$公式是需要Markdown Math插件的，至于公式的许多用法请参考其他问题。
  5. 在公式环境中，不要轻易分行，容易导致公式渲染失败。
